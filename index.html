FAQ

Questions IT always asks.

Straight answers to the questions that come up in every security review.

Where is our data stored?+
100% on your own server. EngiIntel stores all documents, embeddings and query logs in a PostgreSQL database running in Docker on your hardware. No data is ever sent to any external service.
Does it work without internet?+
Yes. After initial setup and model download, EngiIntel operates entirely offline. It can be deployed on an air-gapped intranet with no outbound internet access required.
What hardware do we need?+
For small teams (1–10 users): a modern workstation with 32GB RAM and a mid-range GPU (8GB+ VRAM) runs comfortably. For larger deployments a server with GPU is recommended. Runs on Windows, Linux and macOS via Docker.
Is it GDPR compliant?+
Yes, by architecture. Since all data stays on your infrastructure, there is no cross-border data transfer. Includes a GDPR right-to-erasure endpoint, PII detection, and full audit trail export.
What happens if the server goes down?+
EngiIntel is a standard Docker application. Your IT team manages backups, redundancy and uptime using your existing infrastructure procedures. Enterprise plans include deployment assistance and SLA documentation.
