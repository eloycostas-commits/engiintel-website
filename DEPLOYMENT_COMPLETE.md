## 🔍 Troubleshooting

### If Vercel Deployment Fails:
1. Check Vercel dashboard for error messages
2. Verify build logs
3. Check if index.html has syntax errors
4. Try redeploying manually

### If Website Looks Wrong:
1. Hard refresh browser (Ctrl+F5 or Cmd+Shift+R)
2. Clear browser cache
3. Test in incognito mode
4. Check browser console for errors

### If Tabs Don't Switch:
1. Check browser console for JavaScript errors
2. Verify `switchTab()` function exists
3. Check if tab IDs match button onclick values
4. Test in different browser

### If Screenshots Don't Load:
1. Verify file paths in `screenshots/` folder
2. Check filenames match exactly (case-sensitive)
3. Verify files were pushed to GitHub
4. Check Vercel deployment includes screenshots
